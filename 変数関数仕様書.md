# 変数関数仕様書

## 1. モジュール構成
- `main.rs`: エントリポイント、MCPサーバーハンドラ、CLI定義
- `notifier.rs`: 通知送信ロジック（OS, Discord）
- `config.rs`: 設定管理（tomlファイル, 環境変数）

## 2. 関数・構造体仕様

### 2.1 main.rs

#### OshiraseHandler (struct)
- **役割**: MCPプロトコルのリクエストを処理するハンドラ。
- **フィールド**:
  - `config: Arc<Config>`: アプリケーション設定。
- **依存関係**: `Config` (config.rs)

#### handle_method (method)
- **役割**: MCPメソッド（`tools/list`, `tools/call`）を処理する。
- **引数**:
  - `method: &str`: メソッド名
  - `params: Option<Value>`: パラメータ
- **依存関係**: `notifier.rs`, `Config`
- **影響範囲**: MCPクライアント（Gemini等）へのツール提供

### 2.2 notifier.rs

#### send_os_notification (function)
- **役割**: ローカルOSのデスクトップ通知を送信する。
- **引数**:
  - `title: &str`: 通知タイトル
  - `message: &str`: 通知本文
- **依存関係**: `notify-rust` (crate)
- **影響範囲**: `main.rs` (handle_method, CLI Commands)

#### send_discord_notification (async function)
- **役割**: Discord Webhookを使用して通知を送信する。
- **引数**:
  - `webhook_url: &str`: Discord Webhook URL
  - `content: &str`: 通知本文
  - `prompt: Option<&str>`: 付加情報（プロンプト等）
  - `default_title: &str`: デフォルトタイトル
- **依存関係**: `reqwest` (crate), `chrono` (crate)
- **影響範囲**: `main.rs` (handle_method, CLI Commands)

### 2.3 config.rs

#### Config (struct)
- **役割**: アプリケーション全体の設定を保持する。
- **フィールド**:
  - `discord: DiscordConfig`: Discord関連設定
  - `general: GeneralConfig`: 全般設定
- **依存関係**: `serde`, `toml`

#### Config::load (static method)
- **役割**: 設定を読み込む。環境変数を最優先し、次に `config.toml`、最後にデフォルト値を採用する。
- **戻り値**: `Config`
- **依存関係**: `directories-next`, `fs`, `env`
- **影響範囲**: `main.rs`
